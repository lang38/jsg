使用说明
功能介绍：使用Windows AD域中帐号与密码登录记事狗微博系统！
Windows AD域登录
开启Windows AD域登录: {$ldap_enable} 开启后可用Windows AD域中帐号与密码登录
请输入您的windows AD域帐号: 避免您使用域帐号登录后，无法进入后台
开启该功能后，系统将禁止新用户注册，禁止用户修改密码与修改E-mail，不支持密码找回功能，且系统中原有帐户将无法登录，此时只接收windwos AD域用户登录。
参数设置
Windows AD域服务器地址: 请填写服务器主机名或IP地址（例如：ad.cenwor.com或10.3.58.88）
Windows AD域服务器端口: 默认是：389
